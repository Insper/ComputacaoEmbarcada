!!! info Você deve ter instalado o software Microchip Studio

!!! note O vídeo faz referência ao repositório:

https://github.com/Insper/SAME70-examples

[Acesse o canal clicando aqui!](https://teams.microsoft.com/l/channel/19%3a274b9061bb5143f38d1c79397ab9f536%40thread.tacv2/Suporte?groupId=fce5a025-4419-4faf-8af0-8f5d97bc39fd&tenantId=6370a6c0-7b90-4709-bd6e-59c28ede833b)

Passos para testar:

  1. Conectar OLED1 ao EXT1
  2. Ligar o USB na porta DEBUG
  3. Clonar repositório: https://github.com/Insper/SAME70-examples
  4. Abrir projeto Screens/OLED-Xplained-Pro-SPI/OLED-Xplained-Pro-SPI
  5. Programar microcontrolador.
